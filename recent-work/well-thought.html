"write a blog post about Tweet Thought Ranker, make it seem disruptive"

Tweet Thought Ranker is a Chrome extension that leverages an LLM to rank tweets on a scale of 0 to 10 based on user defined criteria (e.g., creativity, uniqueness, reflectiveness, thoughtfulness, etc.). The main idea behind it is to increase feed browsing experience by focusing on more meaningful content.

how it started

I was listening to Lex Fridman's podcast with Pieter Levels, where Pieter mentioned using gpt-4 to moderate and fact-check information on his communities. Lex shortly commented on how interesting and pontentially more feasible would it be to rate ideas based on how well thought out / how well argued they are. Levels then talked about doing something similar for Twitter replies, and how Elon could actually use Grok to create a better ranking system for the replies.

This discussion sparked me on how fun it would be to have an llm judging twitter posts and hopefully helping to find actual meaningful content. With that goal in mind, I decided to take the Chrome extension route as 1. I never built a chrome extension before and 2. I don't see any other way of doing it. The second point is a bit pity since most people use Twitter on mobile and the extension is completely useless there.

how I did it
